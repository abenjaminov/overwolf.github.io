Here, you can manage the production channel of your app. It is the default version that a user will get,
unless they are specifically subscribed to a different [**App Channel**](./testing).
![The Production Tab](/assets/tools/developers-console/all-applications/release-management/production/index.jpg)

For information on how to manage an app channel, click [here](.#manage-an-app-channel).
