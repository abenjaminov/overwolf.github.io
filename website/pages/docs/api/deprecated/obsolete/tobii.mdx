:::warning OBSOLETE
The overwolf.tobii API is an historical API that would allow the app to interact with the tobii service.  
If you want to integrate tobii in your OW app, please wrap the [tobii DLL](https://developer.tobii.com/consumer-eye-trackers/core-sdk/) as an [OW plugin](/topics/using-plugins/plugins-overview).
:::
